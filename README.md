# koydo-upsc-cse-prelims

UPSC CSE Prelims exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM045 |
| Name | UPSC CSE Prelims |
| Country | India |
| Category | Government/civil service |
| Tier | Tier 2 |
| Wave | Wave 3 |
